# **App Name**: MedMaster AI

## Core Features:

- AI Medical Assistant: AI-Powered Explanations: Utilizes generative AI tool to provide evidence-based explanations, including source citations for medical questions.
- Daily Quizzes: Daily Quizzes: Presents a daily set of medical questions.
- SRS Review:  spaced repetition system with scheduled review dates and automated updating as user uses the product
- Secure Authentication: User Authentication: Secure user authentication via Supabase.
- 3D Anatomy Models: Interactive 3D Anatomy Models: Interactive 3D models with layer toggles and hover labels
- PDF Support: PDF Annotation: Support the user's upload of PDF documents with integrated annotations.
- Offline Capability: Offline Support and Sync: Ensure notes and annotations persist offline.

## Style Guidelines:

- Primary color: Deep Blue (#1E3A8A), evokes trust and intelligence, appropriate for medical content.
- Background color: Light Grey (#F8FAFC), provides a clean, neutral backdrop that is easy on the eyes, appropriate for long hours of study.
- Accent color: Sea Green (#14B8A6), offers a vibrant and modern contrast while still feeling natural and calm.
- Body font: 'PT Sans' (sans-serif), suitable for comfortable reading in longer paragraphs and body text. Headline font: 'Space Grotesk' (sans-serif), a clean and modern sans-serif for headlines.
- Simple and clear icons, using a consistent style (filled or outlined) throughout the app, ensure affordance while contributing to an uncluttered visual environment
- Clean and intuitive layout with a focus on readability. Use of whitespace to separate sections and elements, and avoid clutter.
- Subtle animations for transitions and interactions to provide visual feedback without being distracting.
